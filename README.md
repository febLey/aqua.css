# aqua.css

## Usage

Add the following code into the `<head>` tag of your website:

```html
<link rel="stylesheet" media="all" href="/aqua.css/css/aqua.min.css">
```

or for development:

```html
<link rel="stylesheet" media="all" href="/aqua.css/css/aqua.css">
```
